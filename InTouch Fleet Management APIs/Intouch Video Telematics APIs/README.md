# Intouch Video Telematics APIs
This `Intouch Video Telematics APIs` provide a set of APIs to access video and event information from camera-enabled telematics devices. These APIs allow applications to retrieve live video embed URLs, access historical video embed URLs for recorded footage, and fetch video telematics alarm logs generated by ADAS, DMS, and other camera-based safety systems.

By integrating these APIs, developers can build solutions for real-time vehicle monitoring, historical video review, and safety event analysis within telematics and fleet management platforms.
